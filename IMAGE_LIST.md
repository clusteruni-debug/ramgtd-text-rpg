# 필요한 이미지 목록

## 이미지 넣을 폴더
```
public/images/bg/          ← 배경 이미지
public/images/characters/   ← 캐릭터 이미지
```

> 이미지를 넣으면 코드에서 CSS나 JS로 연결하는 작업은 제가 해드릴게요.
> 권장 사이즈: 배경 800x600, 캐릭터 200x400 (투명 PNG)
